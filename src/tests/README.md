# Каталог tests

Предназначен для хранения исходных текстов тестовых обработок (xUnit / Vanessa-ADD), созданных на платформе 1С:Предприятие.

На каждую тестовую обработку создается отдельный подкаталог с именем обработки. Внутри каталога выкладываются исходники этой обработки (формат разборки `vrunner decompileepf`).

Собранные бинарные файлы тестов (`.epf`) попадают в `build/out/tests` (артефакт сборки, в git не хранится) — их запускает `vrunner xunit`. Сборка из исходников: `vrunner compileepf src/tests build/out/tests` или команда «Собрать unit тесты» расширения 1C: Platform Tools.

Скриптовые тесты OneScript (`*.os`) к этому каталогу не относятся — они лежат сразу в `tests` и запускаются `1testrunner`.
